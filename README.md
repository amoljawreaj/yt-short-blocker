# YouTube Shorts Blocker

A lightweight Chrome extension that completely blocks YouTube Shorts by:
- Redirecting `/shorts` URLs
- Removing Shorts UI elements
- Handling YouTubeâ€™s SPA navigation

## Installation
1. Clone or download this repo
2. Open `chrome://extensions`
3. Enable **Developer Mode**
4. Click **Load unpacked**
5. Select the project folder

## Compatibility
- Chrome / Chromium browsers
- Manifest V3
